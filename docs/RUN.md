# Run checklist:
    * [ ] Activate the virtual environment
    * [ ] Run the static server
    * [ ] Run the app

## Activate the environment
On Windows:
```shell
.venv\Scripts\activate.bat
```

On Mac or Linux:
```shell
.venv/bin/activate
```

## Run the static server
```bash
npm start
```
## Run the app
```bash
python manage.py runserver 8001
```
